# Music Files Directory

This directory contains audio files for the Music Suggestions feature and background music.

## Required Files

Please add the following MP3 files to this directory:

### Mood Booster Playlist
- `Untitled.mp3`
- `Mood Booster.mp3`

### Chill Vibes Playlist
- `Moonlight in My Room.mp3`
- `Midnight Whispers.mp3`
- `Midnight Echoes.mp3`

### Background Music
- `Wellness Piano Loop.mp3` - Plays continuously in the background (can be toggled in settings)

## Note
Make sure the file names match exactly as listed above for the playlists and background music to work correctly.
