## Data Storage

All user data is securely stored in Supabase backend:
- User profiles and preferences
- Daily check-ins and mood history
- Journal entries (password-protected)
- Period tracking data
- Emergency contacts
- Streak records

## Privacy & Security

- Soul Notes are password-protected
- All data is user-specific and isolated
- No data sharing between users
- Secure backend API

## Usage

1. Start with the fun welcome screen
2. Enter your name and gender
3. Complete your first daily check-in
4. Explore wellness tools and features
5. Use the chatbot for 24/7 support

This is a prototype/demo application. For serious mental health concerns, please consult with professional healthcare providers.
